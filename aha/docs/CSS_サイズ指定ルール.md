# CSS サイズ指定ルール

本プロジェクトでは PandaCSS の `css()` でサイズを指定する際、以下のルールに従う。

## 基本方針

**「基本は rem、レスポンシブな幅には %、固定したい箇所だけ px」**

## 単位の使い分け

### フォントサイズ — `rem`

ブラウザの文字サイズ設定に連動させるため、原則 `rem` を使う。
`px` で固定するとユーザーが文字サイズを変更しても反映されない。

```ts
// OK
css({ fontSize: "0.875rem" })

// NG
css({ fontSize: "14px" })
```

> 基準: 1rem = 16px（ブラウザデフォルト）

### レイアウトの幅 — `%` / `vw` / `vh`

画面サイズに応じて伸縮させたいコンテナやカラムには `%` を使う。
画面全幅に広げたい場合は `vw` も可。

```ts
// OK
css({ w: "full" })  // PandaCSS トークン = 100%
css({ maxW: "87.5rem" })  // コンテナの上限幅

// NG
css({ w: "1400px" })
```

### 余白（margin / padding） — `rem` または PandaCSS トークン

文字サイズに連動させたい余白は `rem`。
PandaCSS のスペーシングトークン（`"2"`, `"4"`, `"6"` 等）は内部で rem に変換されるため、そのまま使ってよい。

```ts
// OK — トークン（内部で rem に変換される）
css({ py: "4", px: "6" })

// OK — 明示的な rem
css({ mb: "1.5rem" })

// NG
css({ mb: "24px" })
```

### 最小幅・最大幅 — `rem`

テーブルのカラム幅やフォーム入力欄の最小幅など。

```ts
// OK
css({ minW: "7.5rem" })

// NG
css({ minW: "120px" })
```

### 境界線（border） — `px`

線の太さはデバイス解像度に関わらずくっきり表示させたいため `px` を使う。

```ts
// OK
css({ border: "2px solid transparent" })
css({ borderBottom: "1px solid #c4c5d5" })
```

### メディアクエリ — `px`

ブレークポイントの指定には `px` を使ってよい。

```css
@media (min-width: 768px) { ... }
```

## PandaCSS トークンとの関係

PandaCSS のスペーシングトークン（`"1"` = 0.25rem, `"2"` = 0.5rem, `"4"` = 1rem 等）は内部で rem に変換されるため、トークンが存在する値はトークンを優先して使う。
トークンにない中間値やカスタム値のみ `"1.5rem"` のように明示する。
